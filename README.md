# Pokemon API

Retrieve information about Pokemons, such as their Shakespearean description

### Prerequisites

* Visual Studio 2017
* .NET Core SDK 3.1

### Swagger

https://localhost:44338/swagger/index.html

### Curl Request

curl -X GET "https://localhost:44338/pokemon/charizard" -H "accept: */*"
